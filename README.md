# nvim

### Usage:

- Code Editing
- Note Taking

## Requirements

- Typescript
- Vim with nodejs support __see `./bin`__
- Vim with Ruby Support __see `./bin`__
- Vim with Python support __see `./bin`__

### Features

- Color setup with Base16 and pinnacle
- Sensible defaults
- Hackable structure

### Plugins List

- CommandT: Fuzzy Finder that doesn't suck. (Nav files/buffers)
